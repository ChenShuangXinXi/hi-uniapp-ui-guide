# 统一标签表现形式

HiUi 简单的统一了标签的表现形式。

```scss
page,
view,
scroll-view,
swiper,
swiper-item,
match-media,
movable-area,
movable-view,
cover-view,
cover-image,
icon,
text,
rich-text,
progress,
button,
checkbox-group,
checkbox,
editor,
form,
input,
label,
picker,
picker-view,
radio-group,
radio,
slider,
switch,
textarea,
navigator,
animation-view,
audio,
camera,
image,
video,
live-player,
live-pusher,
map,
canvas,
web-view,
ad,
official-account,
open-data,
barcode,
list,
cell,
recycle-list,
waterfall,
refresh,
::before,
::after {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

image {
    font-size: 0;
    will-change: transform;
}
```
