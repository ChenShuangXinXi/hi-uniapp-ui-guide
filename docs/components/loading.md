# 基础使用

```vue
<hi-loading></hi-loading>
```

<br/>

# 提示文本

通过 `text` 设置 `loading` 提示文本。

```vue
<hi-loading text="加载中..."></hi-loading>
```

<br/>

# Props

| 参数       | 说明               | 类型     | 默认值      | 可选值 |
| ---------- | ------------------ | -------- | ----------- | ------ |
| `text`     | `loading` 提示文字 | `String` | ``          | -      |
| `iconName` | 图标名称           | `String` | `__loading` | `-`    |
